Undergraduate Programmer

C++ | QT | Point Cloud Data | Visual Studio

August 2015 - May 2016, August 2016 - Present

Currently, I work as a student programmer for the vizHOME project here in Madison. This project is a part of the Living Environments Lab, a virtual reality lab located at the Wisconsin Institute for Discovery.

My responsibilities include:

  • Improving the efficiency of the pipeline from lidar scan data to point cloud models for representation in the CAVE.
  • Assisting in research in a technical role, helping participants get comfortable with navigation in our CAVE and executing various experiments and scenarios.
  • Volunteering in outreach events like Science Fest, lab tours, and virtual reality demos.

The vizHOME project focuses on assessing peoples personal health information management using virtual reality to simulate home environments. We have 20 point cloud models of real homes that were collected using a lidar scanner. These models, after processing, can be displayed in our virtual reality CAVE, a 10'x 10'x 10' cube that projects a virtual environment on all sides (See link above for more info)
